 ### Almir Collections
 *a custom built product showcase and reselling platform. i developed this from scratch to manage sales online with the focus on speed and a clean look.*

### Features
- **Products :** users can browse items via categories, search for specific product, and check out trending sections.
- **Detailed View :** dedicated pages for every product to show all the details of the product.
- **Social Proof :** a built in reviews and FAQ section to build trust with customers.
- **Admin Dashboard :** a private area for me to add edit or delete products and reviews in real time.

### Tech Stack
- **Frontend :** react.js and tailwind Css (built from scratch).
- **Database And Auth :** firebase (for the data and secure admin login).
- **Images :** cloudinary (for product photos).

### Setup

1. clone the project
   
```bash
git clone https://github.com/Aflah-Paruthicode/Almir_Collections.git
```

2. navigate into the folder

```bash
cd ./Almir_Collections
```

3. install dependencies

```bash
npm install
```

4. start developing

```bash
npm run dev
```
